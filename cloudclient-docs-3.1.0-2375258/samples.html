Samples

The samples directory in the extracted CloudClient directory contains the following sample scripts. These sample scripts are for a Linux or MacOSX environment.

Script Name Description
ApproveRequest.shApproved/Reject submitted requests
DeployAmazonServer.shProvision an Amazon instance
DeployVSphereServer.shProvision an vSphere instance
ListCatalogItems.sh.shOutputs Catalog Items a user has access to
ListMachines.sh.shOutputs machines a user has access to
PowerOffMachine.shPower Off a specified machine
StartVCOWorkflow.shStart a VCO Workflow
GetVSphereReservationSchema.shGet vSphere reservation schema
GetComputeResourceFieldValues.shGet Compute resources from available vSphere endpoints
GetVSphereReservationFieldsValues.shGet values for all fields in a vSphere reservation schema
GenerateVSphereReservationTemplate.shGenerate a sample json file to use for creating a vSphere reservation
CreateVSphereReservation.shCreate a vSphere reservation
