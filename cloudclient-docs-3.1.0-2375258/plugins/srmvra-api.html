SRM VRA Commands


DescriptionCommand

Register machine into the vRA Inventory

srm vra failover machine

Parameter--name (Machine Name)
Mandatorytrue

Parameter--moRef (Managed Object Reference or unique vCAC ID)
Mandatorytrue

Parameter--export (Export to File)
Mandatoryfalse

Parameter--format (Format the response in either Tabular format (default), JSON or CSV)
Mandatoryfalse

Parameter--clusterName (Name of cluster machine resides on)
Mandatoryfalse
